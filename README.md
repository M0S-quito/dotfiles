# M0S_quito dotfiles
> Personal Linux development environment configuration.

---

## Supported
- Linux
- Ubuntu 24.04 LTS
- WSL2 Ubuntu
- Shell
- Bash

## Included
- Shell configuration
- Git configuration
- Tmux configuration
- Neovim configuration
- Bootstrap scripts
- Setup
- ./setup.sh

## Structure
- config/     # Application configurations
- install/    # Package installation scripts
- shell/      # Shell settings and utility scripts
- setup.sh    # Environment setup entry point

## Install
```
```
git clone https://github.com/M0S-quito/dotfiles.git
bash dotfiles/setup.sh
```

```
